# LATAM Broker Review Projects

This directory contains 10 different broker review websites, each with unique themes, colors, and focus areas.

## Projects Overview

1. **RankingBrokersLatam.com** - Leaderboard/Top 10 style
2. **InversionistaDigital.net** - Modern tech-forward
3. **ZonaTradingLatam.com** - Active trading zone
4. **BrokersConfiables.com** - Safety/trust focused
5. **RutaDeTrading.com** - Educational journey
6. **ComparativaForex.com** - Forex niche
7. **PlataformasDeInversion.com** - Broad investment
8. **GuiadeBrokersSudamerica.com** - Southern Cone specific
9. **TopBrokersMexicoYMas.com** - Mexico focused
10. **CapitalLatamReviews.com** - Institutional/professional

## Theme Configuration

Each project has:
- Unique color scheme (defined in CSS variables)
- Customized layout and structure
- Focused content areas
- Brand-specific navigation

## Building Projects

Each project can be built independently using the main build script with project-specific configuration.
